How the Japanese type using QWERTY keyboard

Three scripts

  1. Hiragana
  2. Katakana
  3. Kanji (not necessary)

So how do they type it?

Keystrokes that map to the sound (In Japanese)

さ = sa く= ku ら = ra
Sakura = さくら

さ = sa よ = yo な = na ら = ra
Sayonara = さよなら

こ = ko に = ni ち = chi わ = wa
Konichiwa = こにちわ
